API Reference

Convert URL to Markdown

Send a GET request with the URL you want to convert:

GET /api/markdown?url=https://example.com

Response

{
"url": "https://example.com",
"title": "Example Domain",
"excerpt": "This domain is for use in illustrative examples...",
"wordCount": 42,
"markdown": "# Example Domain\n\nThis domain is...",
"contentLength": 1250,
"fetchedAt": "2026-07-06T12:00:00.000Z"
}

Authentication

For higher limits, include your API key in the header:

curl -H "x-api-key: md_live_xxxxx" \
  https://api.markdownapi.com/api/markdown?url=https://example.com

Error Codes

400 - Missing or invalid URL
401 - API key required
403 - Invalid or expired API key
429 - Rate limit exceeded
502 - Failed to fetch the URL
504 - Request timed out
